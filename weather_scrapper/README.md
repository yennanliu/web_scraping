


### Quick start

```bash
# get repo and install packages 
$ git clone https://github.com/yennanliu/web_scraping
$ cd web_scraping 
# install library
$ source setup.sh
# run demo script
# export pythonpath 
# export PYTHONPATH=/Users/yennanliu/web_scraping
$ export PYTHONPATH=<ur_file_url>
# export db_url 
$ export db_url=<ur_db_url>
$ python weather_scrapper/br_weather_scrapper_V1.py 

```
